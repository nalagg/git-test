	first pull down master, any changes from other dev will be here
	origin: local
	master: branch

	< FLOW >
	create branch from where you will be working
git branch my-branch-name

	then push that branch, not the entire master
	the u defines the push/pull area
git push -u origin my-branch-name	

	so that in the future you can simply do this and it knows where
git pull
git push
